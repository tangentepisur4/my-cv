# CV responsive

CV responsive bilingue de Jonathan Sande Loh, prêt à être publié gratuitement avec GitHub Pages.

## Publication

Le site public se trouve dans le dossier `docs`.

Pour publier sur GitHub Pages :

1. Envoyez ce projet dans un dépôt GitHub public.
2. Allez dans `Settings` > `Pages`.
3. Choisissez `Deploy from a branch`.
4. Sélectionnez la branche `main` et le dossier `/docs`.
5. Enregistrez.

L'URL finale aura ce format :

```text
https://votre-utilisateur.github.io/nom-du-depot/
```

Voir aussi [README_GITHUB_PAGES.md](README_GITHUB_PAGES.md).
