# Publication du CV sur GitHub Pages

Ce dépôt peut publier le CV gratuitement avec GitHub Pages.

Le site statique à publier se trouve dans le dossier `docs`.

## Réglage GitHub Pages

1. Ouvrir le dépôt GitHub `tangentepisur4/my-cv`.
2. Aller dans `Settings` > `Pages`.
3. Dans `Build and deployment`, choisir :
   - Source: `Deploy from a branch`
   - Branch: `master`
   - Folder: `/docs`
4. Cliquer sur `Save`.

Après quelques minutes, le CV sera disponible ici :

```text
https://tangentepisur4.github.io/my-cv/
```
